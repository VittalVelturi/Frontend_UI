Box-sizing Property

Definition: Defines how the width and height of an element are calculated: should they include padding and borders, or not.

box-sizing: content-box (default):

The width and height properties (and min/max properties) includes only the content. Border and padding are not included

This div has a width of 300px.
Actual Width=Width+Border(L&R)+Padding(L&R)
Actual Width=300+4+60=364px

box-sizing: border-box:

The width and height properties (and min/max properties) includes content, padding and border

Here, the full width is 300px, no matter what!
